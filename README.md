# 🎓 Student Performance Prediction (ML Project)

## 📌 Overview
This project predicts student performance using Machine Learning.

## 🚀 Features
- Data preprocessing
- Feature engineering
- Model training (Linear Regression, Random Forest)
- Model evaluation (R2 Score, MSE)

## 📊 Dataset
- Kaggle: Students Performance in Exams

## 🧠 Models Used
- Linear Regression
- Decision Tree
- Random Forest

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## 📈 Results
- Achieved high accuracy using Random Forest

## ▶️ How to Run
1. Open Jupyter Notebook
